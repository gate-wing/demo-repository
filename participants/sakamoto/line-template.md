# グループLINE送信テンプレート

## 【案内用】

```
お世話になっております！
GMAの〇〇(名前)です。
定例オンラインMTGのご案内です。

【面談日時】
●/●(曜日) ●時〜
参加URL：（ここにMeetのリンクを貼る）

ご都合つかなくなったり、何かあればお気軽にご連絡ください。
よろしくお願いいたします！
```

---

## 【リマインド用】

```
お世話になっております！
GMAの〇〇(名前)です。
明日の定例オンラインMTGのリマインドです。

【面談日時】
●/●(曜日) ●時〜
参加URL：（ここにMeetのリンクを貼る）

ご都合つかなくなったり、何かあればお気軽にご連絡ください。
よろしくお願いいたします！
```

---

## 使い方

1. スマホのメモ帳（iPhoneなら「メモ」、Androidなら「Googleキープ」）に上のテンプレートを保存する
2. 送るときは●の部分（日時・URL）だけ書き換える
3. LINEグループにコピペして送信
