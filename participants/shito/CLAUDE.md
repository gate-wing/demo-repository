# 請求書の自動返信について

## やりたいこと

`kanri@` に届くメールのうち、**件名に「請求書」が含まれるメール**に対して、
**受領確認の返信**を自動で送る。

---

## 使うツール

**Power Automate**（パワーオートメイト）
- Microsoft が提供する自動化ツール
- プログラミング不要で、画面をクリックするだけで設定できる
- Microsoft 365 に含まれているので、追加費用なしで使える

---

## 設定手順

### ステップ1：Power Automate を開く

1. ブラウザで [https://make.powerautomate.com](https://make.powerautomate.com) にアクセスする
2. `kanri@` のアカウントでログインする

### ステップ2：新しいフローを作る

1. 左メニューの「**作成**」をクリックする
2. 「**自動化したクラウド フロー**」を選ぶ
3. フロー名に「請求書自動返信」と入力する
4. トリガー（きっかけ）として「**新しいメールが届いたとき**」を選ぶ
5. 「**作成**」をクリックする

### ステップ3：条件を設定する（件名に「請求書」が含まれるメールだけ）

1. 「**新しいメールが届いたとき**」の設定画面で、「件名フィルター」に `請求書` と入力する

### ステップ4：返信内容を設定する

1. 「**＋ 新しいステップ**」をクリックする
2. 「**返信する（返信の送信）**」を選ぶ
3. 返信本文に以下のような文章を入力する：

```
いつもお世話になっております。

請求書を受領いたしました。
内容を確認のうえ、処理いたします。

どうぞよろしくお願いいたします。
```

### ステップ5：保存して有効にする

1. 右上の「**保存**」をクリックする
2. フローが有効になっていることを確認する

---

## 注意事項

- 同じ相手に何度も自動返信が送られないよう、Power Automate が自動で制御してくれる
- 設定後はテスト用のメールを送って、正しく返信が届くか確認することをおすすめする
