# 労働分配率・適正人数 分析の指示書

## 会社情報

- 従業員数：21名
- 業種：外食向けWEB販促支援（HP作成・外食媒体販売）

---

## STEP 1：CSVファイルを共有する

試算表のCSVファイルをClaudeに送る。

**確認してほしい項目：**

| 項目 | 勘定科目の例 |
|------|------------|
| 売上高 | 売上、売上高、営業収益 など |
| 人件費 | 給料、給与、賞与、法定福利費、福利厚生費 など |
| 粗利益 | 売上総利益（もしあれば） |

---

## STEP 2：労働分配率を計算する

CSVを受け取ったら、以下の計算を行う。

### 計算式

```
労働分配率(%) ＝ 人件費合計 ÷ 粗利益 × 100
```

※ 粗利益がCSVにない場合は以下で計算する：
```
粗利益 ＝ 売上高 ー 売上原価
```

※ 売上原価がない業種（サービス業など）の場合：
```
粗利益 ＝ 売上高 をそのまま使う
```

---

## STEP 3：業種別の適正労働分配率と比較する

### 外食WEB販促支援業の目安

| 指標 | 目安 |
|------|------|
| 適正な労働分配率 | **50〜60%** |
| 要注意ライン | 60〜70% |
| 危険ライン | 70%以上 |

※ WEB・広告・販促支援はサービス業に近いため、人件費の割合が高くなりやすい。

---

## STEP 4：適正従業員人数を割り出す

### 計算式

```
適正人件費 ＝ 粗利益 × 適正労働分配率(55%を基準)
適正人数 ＝ 適正人件費 ÷ 従業員1人あたりの平均人件費
```

### 現状との比較

| 項目 | 数値 |
|------|------|
| 現在の従業員数 | 21名 |
| 計算上の適正人数 | CSVをもとに算出 |
| 過不足 | 適正人数との差を表示 |

---

## STEP 5：結果をまとめる

CSVを受け取った後、以下の形式でまとめて報告する。

```
【分析結果】

■ 売上高：〇〇〇万円
■ 粗利益：〇〇〇万円
■ 人件費合計：〇〇〇万円

■ 労働分配率：〇〇%
　→ 判定：適正 / 要注意 / 危険

■ 適正従業員人数：約〇〇名
　→ 現在21名に対して 〇名 多い / 少ない / ほぼ適正
```

---

## 注意事項

- 試算表の期間（月次・年次）を確認してから計算する
- 年次でない場合は年換算して比較する
- 役員報酬が人件費に含まれているか確認する（含まれている場合は分けて計算する）
