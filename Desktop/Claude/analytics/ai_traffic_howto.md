# AIエンジン参照トラフィック 横断レポート 実行手順

ChatGPT・Gemini などのAIエンジンから、各店舗サイトへの訪問がどれくらいあるかを  
全プロパティ横断で確認するためのレポートを作成します。

---

## 必要なもの

- このフォルダに `credentials.json` と `token.json` があること
- Python がインストール済みであること

---

## 実行手順

### Step 1: データを取得する

ターミナルで以下を実行してください。

```
python -X utf8 fetch_ai_traffic.py
```

**何が起きるか**

1. Google アカウントの認証（初回のみブラウザが開きます）
2. 「株式会社ぐっとくるダイニング」の全プロパティを自動で検出
3. 各プロパティの直近12ヶ月分のAI参照データを取得
4. `ai_traffic_data.json` というファイルに保存

**実行中の表示例**

```

    プロパティ: 3B神保町 (ID: 326417963)
    プロパティ: ○○店   (ID: xxxxxxx)
  合計 2 プロパティを対象とします。

▶ 分析期間: 2025-04 〜 2026-03 (12ヶ月)

  2025-05: 取得中...

```

---

### Step 2: レポートを生成する

```
python -X utf8 export_ai_report.py
```

**何が起きるか**

`ai_traffic_data.json` を読み込んで、以下の2ファイルを生成します。

| ファイル名 | 内容 |
|-----------|------|
| `ai_traffic_report_YYYYMM.md` | テキスト形式のレポート |
| `ai_traffic_report_YYYYMM.pptx` | グラフ付きのスライド資料 |

**実行中の表示例**

```
▶ Markdownレポート生成中...
▶ PowerPointレポート生成中...
  対象プロパティ: 2 件
  スライド作成中: タイトル...
  スライド作成中: 全体サマリー...
  グラフ生成中: 月別推移...
  グラフ生成中: エンジン別シェア...
  グラフ生成中: [1/2] 3B神保町...
  グラフ生成中: [2/2] ○○店...
✅ レポートを生成しました。
```

---

## 出力されるスライドの構成

| スライド | 内容 |
|---------|------|
| 1 | タイトル |
| 2 | 全店舗のサマリー（AIエンジン別合計セッション数） |
| 3 | 月別AI参照推移グラフ（全店舗合計） |
| 4 | AIエンジン別シェア（円グラフ） |
| 5 | AIエンジン別の月別トレンド |
| 6 | 店舗別のAI参照合計比較 |
| 7〜 | 各店舗ごとの月別内訳グラフ |

## 対象AIエンジン

以下のサービスからの参照を集計します。  
追加・変更は `ai_traffic_config.yaml` の `ai_sources` 欄を編集してください。

| エンジン名 | ドメイン |
|-----------|---------|
| ChatGPT | openai.com / chatgpt.com |
| Gemini | gemini.google.com |
| Perplexity | perplexity.ai |
| Microsoft Copilot | copilot.microsoft.com |
| Claude | claude.ai |

## うまくいかないときは

**「対象アカウントが見つかりません」と表示される**

`ai_traffic_config.yaml` の `target_account_name` の文字列が  
GA4のアカウント名と一致していない可能性があります。  
以下を実行してアカウント名を確認してください。

```
python -X utf8 fetch_ids.py
```

**ブラウザが開いて認証を求められた（初回）**

Google アカウントでログインして許可するだけで大丈夫です。  
一度認証すると `token.json` に保存され、次回からは不要です。

**「ai_traffic_data.json が見つかりません」と表示される**

Step 1 をまだ実行していません。先に `fetch_ai_traffic.py` を実行してください。
