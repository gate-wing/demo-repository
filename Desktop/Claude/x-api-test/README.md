# X Post Daily Pipeline

X(旧Twitter)のポストを毎日自動で収集し、AIが記事の下書きを作成するワークフローです。

## やっていること

1. **ポストの収集** --- Google Sheetsの「accounts」シートに登録されたXアカウントから、24時間ごとに新しいポストを取得します
2. **分類と記録** --- 取得したポストを「AI・テクノロジー」「ビジネス・仕事」などのカテゴリに自動分類し、「post_logs」シートに記録します
3. **記事の自動生成** --- 直近7日分のオリジナル投稿(3件以上ある場合)をもとに、Gemini APIがnote.com風の記事を下書きします
4. **Googleドキュメントに保存** --- 生成された記事をGoogleドキュメントとして作成し、リンクを「article_drafts」シートに記録します

## 使っているサービス

| サービス | 用途 |
|----------|------|
| X API | ポストの取得 |
| Google Sheets | アカウント管理・ポスト記録・下書き管理 |
| Google Gemini API | 記事の自動生成 |
| Google Docs | 下書きの保存先 |

## 処理の流れ

```
毎日自動実行
  -> Sheetsからアカウント一覧を取得
    -> 有効なアカウントだけに絞り込み
      -> 1件ずつ X API でポストを取得
        -> カテゴリ分類して Sheets に記録
          -> 直近7日のポストから記事を生成(Gemini)
            -> Google Docs に保存 & Sheets に記録
```
